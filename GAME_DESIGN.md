## 1. 最终核心规则

### 1.1 世界模型：怎么算"往下挖"
- 矿层=按需逐行生成的无限竖井网格。消除后上方方块塌落补位，**表面整体下沉=挖掘**。
- **深度(米) = 累计消除方块数 ÷ 8**（等效消一整行=1米）。确定性、好结算。
- 镜头把"当前表面最高行"锁在屏幕第2~3行：`cameraY += (target-cameraY)*0.12`/帧，限速900px/s，到位带8px超调回弹。

### 1.2 布局（750×1334）
| 区域 | 规格 |
|---|---|
| 顶部HUD | 170px：耐久条、深度表、金币、Fever槽、combo计时条 |
| 网格 | 8列×12可视行，90×90px，左右各留15px |
| 底部栏 | 84px：暂停、Buff图标、撤离按钮 |

### 1.3 输入
- 命中：`col=floor((x-15)/90)`，`row=floor((y-cameraY)/90)`；HUD区不落入网格。
- **点击**=位移<24px且<350ms；**拖拽**=位移≥24px（仅特殊道具可拖，普通块拖拽回弹抖动）。
- **按下≤50ms内**：同色连通组高亮描边+放大1.05（预判反馈），抬起时执行。
- **输入缓冲**：消除/下落动画（120~400ms）期间输入进队列不丢，动画永不阻塞操作——手感第一优先级。

### 1.4 点击行为与耐久（局内唯一资源）
| 点击对象 | 行为 | 耐久 |
|---|---|---|
| ≥3同色连通组（4向） | 整组消除 | -1（不论大小→奖励找大组） |
| 1~2连普通块 | 单敲硬掘，直接碎该块（**永不卡死**的保底） | -3 |
| 硬岩🪨 | 单敲直接碎 | -5 |
| 特殊道具 | 立即引爆 | 0 |
| 空地/无效 | 抖3帧，不清combo | 0 |

耐久归零→结算，**收益全额保留无惩罚**。复活每局1次（激励视频桩按钮）：+60耐久+清周围5×5。

### 1.5 特殊道具（四档）与拖拽
| 组大小 | 道具 | 效果 |
|---|---|---|
| 4~5连 | 🚀火箭 | 清整行/整列（组外接矩形高>宽→竖，否则横） |
| 6~8连 | 💣炸弹 | 3×3 |
| 9~11连 | 🌀钻地机 | 沿列下钻15格（+1.875m），沿途碎硬岩免耐久 |
| ≥12连 | 🌈彩钻 | 消全屏最多色 |

拖到另一道具（落点≤0.6格）→合并引爆；拖到普通格→移动+引爆（瞄准）。拖拽中显示范围虚线预览；无效落点200ms回弹。场上道具上限6个（超出最早的自动引爆）。

**合并表（9条）**：🚀+🚀=整行+整列十字；🚀+💣=3行+3列粗十字；💣+💣=5×5；🌀+🚀=下钻15格每5格横扫1行；🌀+💣=下钻15格沿途3×3爆；🌈+🚀=最多色全变火箭0.1s齐射；🌈+💣=随机8处3×3连环（120ms节拍）；🌈+🌀=全列同钻8格（+8m）；🌈+🌈=清空可视区（约+10m）。

### 1.6 连击 / 连锁 / Fever
- **连击**：上次消除后2.5s内再消+1；得分倍率×(1+0.1×combo)上限×3.0；HUD显示衰减计时条。局末记"连击峰值"进金币结算。
- **连锁**：不靠连点，靠埋藏TNT🧨与道具互引——被波及后**固定120ms节拍逐级起爆**（无音效期的音乐代偿）。第k级得分×(1+0.3k)，飘字逐级+4px、白→黄→橙→红。
- **Fever**：槽120点；一次消n块充`n+max(0,n-4)×2`（连锁波及计入）。满槽自动触发，基础8s，Fever中≥5块消除+0.5s，上限12s，结束清零。
  期间：局内金币拾取×2、**耐久全免**、所有消除额外波及1圈、≥2连即可消、道具门槛-1档、岩浆暂停。演出：背景色相滚动+方块脉冲+FEVER呼吸字。
  节奏目标：**80~100秒一次，一局4~6次**（调频只动槽上限）。

### 1.7 死局兜底
可视区无≥3组且无道具→自动"塌方松动"：400ms全屏随机重排+震屏+尘土，免费不限次。兜底做成演出不做成惩罚。

### 1.8 地层与Run曲线（原型固定序列）
| 深度 | 地层 | 引入内容（每层恰好一个新元素） |
|---|---|---|
| 0~30m | 表土层 | 4色零障碍热身；首次Fever约0:50 |
| 30~70m | 岩层 | 硬岩🪨(6%)、TNT🧨埋藏2%；**三选一#1** |
| 70~120m | 冰晶层 | 升5色；冰壳🧊(8%)：第1次波及碎壳第2次消本体；连击≥5自带"热浪"直接破冰（提案3子规则）；三选一#2 |
| 120~180m | 瓦斯层 | 毒块☠️(4%)：被消除感染相邻2块、消毒块-3耐久0金币；**连击达5瞬间全屏毒块净化每块+8金币**（危险变奖励）；三选一#3 |
| 180m+ | 熔岩层→无尽 | **岩浆追击线**从屏顶0.25行/s下压，追上表面-5耐久/秒；此后每+50m：硬岩+2%、毒块+1%、TNT+0.5%(上限5%)、深度分持续涨；三选一#4起每50m一次 |

- 每层末**矿脉房**：一整行金矿块+1个化石块🦴（波及3次出土→矿物图鉴条目+80金币）。
- 每层中段埋1个**事件行**（整行发光+文字），挖到触发（见1.10）。

### 1.9 入层三选一（原型12条，白60%/蓝30%/金10%；耐久<30%保底出"修理"）
1.火箭门槛4→3(金) 2.炸弹3×3→5×5(金) 3.Fever充能+25%(蓝) 4.耐久上限+30并回满(白) 5.combo窗口+1s(蓝) 6.单敲耗3→1(蓝) 7.金币+40%(白) 8.≥8块消除15%掉随机道具(金) 9.硬岩1次波及即碎(白) 10.毒块被道具炸毁+30金币(蓝) 11.连锁每级0.3→0.5(金) 12.里程碑回复10→25(白)
（流派标签+2件套羁绊=二期，池扩到24条）

### 1.11 得分与局末结算
- 单块基础分 `10+深度m×0.2`；一次消n块：`Σbase×(1+0.15×(n-3))×combo倍率×Fever(×2)×连锁(1+0.3k)`。分数只进飘字/纪录/排行榜。
- 金矿块💰出现率：层1为3%每层+1%上限8%；每块=`5+层数×3`金币，且**+1耐久**。
- **局末金币 =（局内拾取金币 + 最深深度m×2 + 连击峰值×2）×（1 + 商店加成 + 藏品加成 + 套装加成 + 皮肤加成 + 图鉴等级×2%）**，当日首局全额×2。

### 1.12 爽感规格（全部文字/色块可实现）
- 方块=圆角色块+中央emoji双编码（色盲友好）：🟥🟦🟩🟨🟪 / 🪨🧨🧊☠️💰🔧📦🦴 / 🚀💣🌀🌈。
- **顿帧**：道具引爆80ms全局冻结，合并大招120ms。
- **分级震屏**：<6块不震 / ≥6块4px·100ms / 炸弹8px·180ms / 大招12px·250ms（衰减正弦偏移canvas矩阵）。
- **阶梯飘字**：普通白24px；大组黄36px弹跳；combo每5级换措辞`不错→漂亮→华丽→疯狂→神级!!`字号32→56px；同屏上限20对象池。
- **振动**：wx.vibrateShort light/medium/heavy 三档映射，节流≥100ms，可关。
- 动画表：消除scale1.2+fade 120ms+4~6碎片粒子(全屏上限120)；下落3000px/s²落地squash；每50m里程碑横幅+heavy震动；结算分数800ms滚动+金币逐枚贝塞尔飞入。
- **SFX事件总线现在就定义**（tap/blast_s/m/l/combo_up/fever_start/end/milestone/merge/coin），播放函数留空。

### 1.13 核心机制备选（已裁决）
主案=点击消除（单次决策<350ms，最匹配高频爽点+单手竖屏）。备选B划线连接：拇指疲劳且与道具拖拽手势冲突，仅二期作特殊层变奏；备选C点击自动钻掘：操作密度太低，弃。
