# 掘掘乐 · 微信小游戏原型

竖屏点击消除式挖掘街机：三连消除挖掘 + 轻rogue三选一 + Fever/连锁爽感 + 局外商店/盲盒/图鉴/转盘/拼图/皮肤。
**贴图已接入真实美术**（assets/，缺图自动回退文字占位）；音效仍为占位，清单见 `ASSETS_NEEDED.md`；设计文档见 `GAME_DESIGN.md`。
切图管线：新原图放 `assets/raw/` → 浏览器开 `tools/slice.html`（配合 `tools/server.ps1`）自动去底/分割/落盘。
UI：Bluestone 像素套件（`assets/bs_*.png` 九宫格：按钮/面板/槽位/图鉴书本）；字体 Xiaolai（`assets/fonts/`）。
⚠️ 微信上线注意：Xiaolai.ttf 约22MB 超小游戏包体限制（主包4MB），需字体子集化（fontmin 等，游戏用字≈500个）或 CDN + wx.loadFontFace。

## 运行

**浏览器调试（推荐日常开发）**
- 任意静态服务器指向本目录即可，例如：`powershell -File tools/server.ps1` 后打开 http://127.0.0.1:8765
- 手机竖屏比例模拟；鼠标点击=手指点击，按住拖动=拖拽

**微信开发者工具**
- 导入本目录（`project.config.json` 已配置为小游戏 compileType=game，appid 为测试号）
- 入口 `game.js`（与浏览器共用同一套 `js/` 代码，双端兼容层在 `js/platform.js`）

## 玩法速览
- 点击 ≥3 个相连同色方块 → 整组消除（耗1耐久）；1~2 连单敲硬掘（耗3）
- 4/6/9/12 连生成 🚀火箭 / 💣炸弹 / 🌀钻地机 / 🌈彩钻；**道具可拖到另一道具上合并**成大招，或拖到任意格瞄准引爆
- 消除充能 Fever：期间免耗、2连即消、波及一圈、金币×2；4% 暴击消除（分×3+金币）
- 每进一个地层三选一强化（16条+四流派羁绊，可"都不选修镐"）；耐久归零结算

## 留存/上瘾/新手/深度（v1.1）
- **每日**：今日矿情（7种日修饰符，联动每日目标）· 每日挑战（2硬修饰符+💎40/盲盒券）· 隔夜矿车（昨日深度=今日开门红）· 明日预告
- **上瘾钩子**：破纪录 near-miss 追赶线 · 撤离倍率 push-your-luck（100m起每50m+10%，主动撤离才拿）· 趁热打铁（连点再来一局赠 🚀→💣→🌀）· 结算"就差一点"诱饵 · 本局高光
- **新手**：前3局必爽包（3连出火箭/充能+25%/单敲减耗/首局3色+6连印章）· 75s强制Fever · 45s合并赠礼 · 四步零文字墙引导 · 发呆自动提示
- **深度**：流派羁绊 build · 修镐拒绝稀释 · 挑战修饰符逼改打法 · 撤离时机决策

## 代码结构
```

js/grid.js         棋盘：连通消除/补位/道具/连锁
js/run.js          单局状态机：耐久/连击/Fever/地层/事件/结算

js/uikit.js        Canvas即时模式UI  js/fx.js 飘字震屏粒子  js/assets.js 资源占位注册表
```

## 调参旋钮（GAME_DESIGN.md §4 备忘）
- 局长 → `data.js tune.layerHeal`（过层耐久回复）
- Fever频率 → `tune.feverMax`
- 经济速度 → `run.js buildSettle` 深度系数、金矿/道具产币在 `run.js handleEvents`
- 抽卡节奏 → 盲盒券发放（每日目标/签到/转盘权重）；开盒为CSGO式横向滚带动画（结果先定、点击任意处直接揭晓，`scene_meta.js startCase/drawCase`）

## 化石收藏（v1.3）
12件三档：**绿·优秀×5**（菊石/三叶虫/古鱼/足印/考古现场）→ **紫·史诗×4**（霸王龙/三角龙/藏宝图/狗头金,40m+）→ **橙·传世×3**（《蒙娜丽鼠》《戴珍珠耳环的鼹鼠》《鼹星夜》,100m+且越深越易出）。
挖到化石块随机出土（60%偏向未收集）；首件给星钻(10/30/80)、重复转星尘；**集齐整档永久加成**：绿=金币+3%、紫=耐久+10、橙=深度结算+5%。图鉴书本为**翻页式**（矿物页在前、化石收藏页在后，底部 上页/下页；未拥有显示灰影+出土深度提示）。配置在 `data.js D.fossilList/rollFossil`。
